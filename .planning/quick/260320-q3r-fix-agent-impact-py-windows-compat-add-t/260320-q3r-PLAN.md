Then update the `run()` helper function (line 34) to suppress stderr at the Python level. Change:
```python
r = subprocess.run(cmd, shell=True, capture_output=True, text=True,
                   cwd=cwd, env=env, encoding="utf-8", errors="replace")
```
to:
```python
r = subprocess.run(cmd, shell=True, stdout=subprocess.PIPE, stderr=subprocess.DEVNULL,
                   text=True, cwd=cwd, env=env, encoding="utf-8", errors="replace")
```

Note: We replace `capture_output=True` with explicit `stdout=subprocess.PIPE, stderr=subprocess.DEVNULL` because `capture_output` sets both stdout and stderr to PIPE, and we want stderr silenced instead.
